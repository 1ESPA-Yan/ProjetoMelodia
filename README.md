# 🎵 App Melodia — Landing Page

Landing page desenvolvida como projeto de avaliação para a disciplina de Front-end Design da FIAP.

## 📋 Sobre o Projeto

O **Melodia** é um aplicativo de música fictício. O objetivo foi desenvolver uma landing page moderna, responsiva e visualmente atraente para divulgar o app, seguindo as orientações da disciplina.

## 🎯 Seções da Página

- **Navbar** — Navegação fixa com logo, links âncora e botão de download
- **Hero** — Título principal, descrição, CTA e imagem do aplicativo
- **Benefícios** — 4 cards com os principais diferenciais do app
- **Funcionalidades** — Galeria com imagens e descrições das funcionalidades
- **Depoimentos** — 3 cards com avaliações de usuários
- **Contato** — Formulário de captura de e-mail
- **Rodapé** — Logo, navegação, redes sociais e informações de contato

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura da página |

## 📱 Responsividade

A página foi desenvolvida com abordagem mobile-first utilizando os breakpoints do Tailwind CSS:

- **Mobile** — até 768px
- **Tablet** — `md:` 768px+
- **Desktop** — `lg:` 1024px+
